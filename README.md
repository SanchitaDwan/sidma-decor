# 🌟 Sidma Decor – AI Powered Lighting Website

A modern, premium lighting e-commerce website with an integrated AI advisor, smart quiz, and seamless shopping experience.

This project helps users **discover, visualize, and buy lighting products** based on their space, mood, and budget.

---

## 🚀 Features

### 🧠 AI Lighting Advisor

* Chat-based interface for lighting recommendations
* Image upload support for personalized suggestions
* Dynamic product suggestions
* Ready for API integration

---

### 🛍️ E-commerce System

* Add to cart functionality
* Real-time cart updates
* Quantity management
* Subtotal calculation
* Cart data stored using LocalStorage

---

### 🎯 Smart Lighting Quiz

* Interactive multi-step quiz
* Personalized recommendations
* Product suggestions based on user preferences

---

### 🏬 Shop Page

* Product listing with filters:

  * Category
  * Price range
  * Product type
* Sorting options (price, popularity, newest)

---

### 🎨 UI/UX Design

* Fully responsive (mobile + desktop)
* Built using Tailwind CSS
* Smooth animations & transitions
* Clean and premium dark theme

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/sidma-decor.git
```

2. Open the folder:

```
cd sidma-decor
```

3. Run the project:

* Open `index.html` in browser
  OR
* Use Live Server (recommended)
